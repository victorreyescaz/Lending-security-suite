# Vulnerable Variant

Esta carpeta está pensada para alojar versiones intencionalmente vulnerables del protocolo, con el objetivo de reproducir PoCs.

Estado actual:
- Coincide con el baseline del MVP (no hay vulnerabilidades intencionales todavía).

Cuando introduzcas una vulnerabilidad:
1. Modifica archivos dentro de esta carpeta.
2. Documenta el finding y el diff en `protocol/CHANGELOG.md`.
3. Enlaza el PoC/test que reproduce el issue.
