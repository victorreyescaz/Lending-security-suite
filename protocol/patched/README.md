# Patched Variant

Esta carpeta está pensada para alojar las versiones corregidas/mitigadas del protocolo correspondientes a `vulnerable/`.

Estado actual:
- Coincide con el baseline del MVP (no hay parches aplicados todavía).

Cuando introduzcas un parche:
1. Aplica el fix aquí.
2. Registra el cambio en `protocol/CHANGELOG.md`.
3. Referencia el PoC/test que el fix resuelve.
